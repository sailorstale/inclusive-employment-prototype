# Инклюзивное трудоустройство — прототип сайта

Статический прототип навигируемого сайта-гида «Инклюзивное трудоустройство»
(бренд «Инклюзия в Яндексе»): практическая информация для трёх аудиторий —
компаний, НКО и соискателей. Собран по постраничной спецификации; весь текст —
из авторского гида.

## Что внутри

- **35 страниц + 404**, три ролевых трека (Компании / НКО / Соискатели) и сквозные
  страницы (глоссарий, «Трудоустройство в Яндексе», обратная связь, доступность).
- Хеш-маршрутизация (`#/...`), боковое меню трека, хлебные крошки, блок «Читайте
  также», пошаговая навигация в воркфлоу найма.
- Светлая и тёмная темы, клиентский поиск по страницам, тултипы-термины со связкой
  с глоссарием.

## Стек

React 18 · Vite · TypeScript · Tailwind CSS · компоненты в духе shadcn/ui
(Radix) · react-router (HashRouter).

## Запуск

```bash
npm install
npm run dev      # дев-сервер (Vite сам выберет свободный порт)
npm run build    # продакшн-сборка в dist/
```

## Структура

```
src/
  components/        канонические компоненты (UI-kit) + ui/ (примитивы) + shell/ (оболочка)
  data/              nav.ts (навигация как данные), glossary.ts (словарь терминов)
  pages/             по одному компоненту на страницу сайта
  globals.css        токены тем (zinc + сдержанный синий акцент)
```

## Объём прототипа

Вне объёма (сознательно): доступность (a11y/WCAG) и мобильная адаптация —
это десктоп-прототип. Часть внешних ссылок и форма обратной связи — заглушки,
помечены как «в работе».
