# Аудит понятности глазами роли

Проход по страницам сайта от лица его аудитории: где текст непонятен ТАК, что рядом (глоссарий, тултип, соседняя страница, следующий шаг) это не решает. Роль берётся из трека: `/companies` → работодатель, `/ngo` → куратор НКО, `/jobseekers` → соискатель, общие → новый посетитель.

**Как устроено (контекст-осознанная версия).** Каждому агенту-читателю дают карту сайта, полный список глоссария (65 терминов + механизм тултипов `<GlossaryTerm>`) и **правило дисконта**: не помечать то, что читатель решает в 1–2 клика (термин из глоссария; «что дальше» через StepNav/«Читайте также»; «где я» через крошки/боковое меню). Настоящая находка навигации — только тупик, когда ответа отсюда нет и ссылки к нему нет. Затем скептик добивает шум, а стадия сжатия сводит сквозные темы в один пункт. Флоу: `.claude/workflows/persona-clarity-audit.js`.

## Три нити на весь сайт

1. **Монобренд Яндекса без честной оговорки.** Контент завязан на Яндекс, но природа ресурса не названа — недоверчивый читатель видит рекламу, а не независимую помощь. Больнее всего соискателю (главный страх «не обманут ли»).
2. **Нет юр-мостика там, где возникает страх.** Работодатель: «квота → штраф → проверка» и «чем грозит нарушение» не связаны ссылкой с правовым разделом ровно в точке тревоги. НКО: юр-опора и разбор упоминаются, но ссылки к `/ngo/legal` в горячих абзацах нет.
3. **Отсылки наружу вместо своего раздела.** В юридически чувствительных местах текст ведёт на внешние Google-доки / «спросите нейросеть», а не на внутренний раздел — подрывает доверие роли, боящейся ошибиться.

## Быстрые победы (подтверждённые баги ссылок)

- `/ngo/candidates` (NgoCandidatesPage.tsx:1448): `<SmartLink to="/legal">` — маршрута `/legal` нет → **404** ровно там, где обещана правовая конкретика. Верный target — `/ngo/legal`.
- `/ngo/reality` (NgoRealityPage.tsx:592): `<SmartLink to="/companies/legal">` уводит из трека НКО в трек работодателя. Должно быть `/ngo/legal`.

Повторный прогон: `Workflow(name: persona-clarity-audit)` — весь сайт, либо `args: "companies"` / `"/ngo/legal"` — трек или страница.
